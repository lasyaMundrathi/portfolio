AWS E-commerce Chatbot: Streamlit + RAG-Powered LLM with CI/CD Deployment

Welcome to the AI-powered chatbot 🤖 built for e-commerce platforms 🛍️. This intelligent assistant helps users with product discovery, real-time support, and answers to store-related questions, improving the overall customer experience.

  • → Built with Retrieval-Augmented Generation (RAG) using Streamlit, Pinecone, and LangChain + Google Generative AI.
  • → CI/CD pipeline using Docker, GitHub Actions, Amazon ECR, ECS (Fargate), and secure key handling via AWS Secrets Manager.
  • → Also deployed on GCP using Artifact Registry, Cloud Run, and GitHub Actions for automated deployment.
View AWS Deployment View GCP Deployment

Architecture

  • Frontend/User Interface: Built with Streamlit, providing a sleek, intuitive UI for users to interact with the chatbot.
    • 🗨️ “What are the key features of this product?”
    • 🗨️ “Which headphones should I buy for good bass?”
  • Backend/Compute: Hosted on Amazon ECS (Fargate), enabling a serverless, fully managed deployment. Utilizes Pinecone for vector search and document embeddings.
  • AI Engine: Combines LangChain and Google Generative AI to generate context-aware, human-like responses.
  • CI/CD Pipeline:
    • 1️⃣ Build Docker container with GitHub Actions
    • 2️⃣ Push to Amazon ECR
    • 3️⃣ Deploy to Amazon ECS (Fargate)
  • Secrets & Monitoring: AWS Secrets Manager injects API keys securely; CloudWatch tracks logs and performance.

Deployment Process

  • Code Management: Project stored in GitHub (frontend, backend, CI/CD).
  • CI/CD Automation: GitHub Actions build and push Docker image to ECR.
  • Deployment: Container deployed to ECS (Fargate) via task definitions.
  • Secrets: API keys injected securely using AWS Secrets Manager.
  • Monitoring: CloudWatch logs system performance and errors in real time.

Challenges Solved

  • 🔍 Fast Query Handling: Pinecone enables real-time semantic search on product reviews.
  • 🌐 Scalability: AWS Fargate allows auto-scaling without manual effort.
  • ⚙️ Automation: GitHub Actions fully automate deployment for rapid iteration.
  • 🔒 Security: AWS Secrets Manager protects all sensitive credentials and APIs.

Outcome

The chatbot delivers a scalable, secure, and intelligent solution for e-commerce platforms, enhancing customer engagement with real-time, contextual assistance while reducing support overhead through smart automation and efficient cloud-native deployment.
